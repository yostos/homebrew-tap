# Homebrew Tap for yostos

This is a [Homebrew](https://brew.sh/) tap for software by yostos.

## Installation

```bash
brew tap yostos/tap
```

## Available Formulae

### ttt (Tiny Task Tool)

A minimal TUI task manager for the terminal.

```bash
brew install yostos/tap/ttt
```

Or install directly without tapping:

```bash
brew install yostos/tap/ttt
```

For more information about ttt, visit: https://github.com/yostos/tiny-task-tool
